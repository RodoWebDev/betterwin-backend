## Description

[BetterWin](https://github.com/nestjs/nest) The world's first real Game-Changer in e-sports and sports-betting.
BetterWin is the first ever platform that drives a tokenized revolution to gaming.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
